

# mindmap-poster


## Purpose

The “mindmap-poster” is a showcase for

* the [XeTeX-engine](http://scripts.sil.org/XeTeX) and

* the (La)TeX-package [TikZ/PGF](http://sourceforge.net/projects/pgf/), in particular the mindmap functionality.


## How to Compile

Just type

    xelatex mindmap.tex
